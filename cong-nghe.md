## 1. Backend

Sử dụng Java Spring Boot để xây dựng API RESTful.

Sử dụng Redis để lưu trữ session.

Sử dụng JWT (JSON Web Tokens) để xác thực và phân quyền người dùng.

Sử dụng WebSocket (Socket.io) để xây dựng tính năng chat nội bộ.

## 2. Frontend

Sử dụng React.js để xây dựng giao diện người dùng.

Sử dụng Redux để quản lý trạng thái ứng dụng.

Sử dụng Tailwind CSS với Framer Motion để xây dựng giao diện người dùng chuyển động mượt mà.

Sử dụng WebSocket (Socket.io) để xây dựng tính năng chat nội bộ.

## 3. Database

Sử dụng MySQL để lưu trữ dữ liệu.

Sử dụng Hibernate để tương tác với MySQL.

## 4. Các công nghệ khác

Sử dụng Git để quản lý mã nguồn.

Sử dụng GitHub để lưu trữ mã nguồn.

Sử dụng Docker để đóng gói ứng dụng.
